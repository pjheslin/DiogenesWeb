DiogenesWeb

DiogenesWeb is an web application, optimized for handheld devices, for reading XML files of Latin and Ancient Greek. These can be open-access files or can be files held in your private cloud storage.

Privacy Notice

DiogenesWeb does not use cookies or store centrally any user information; all user information is stored in your browser. Each time you access DiogenesWeb, the category of user (as identified above) is logged, nothing more.
